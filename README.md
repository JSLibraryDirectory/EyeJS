## Getting Started

To use EyeJS, you'll need to download EyeJS via NPM, and create a `test.js` or `eye.js` file.
```bash
$ npm install eye.js
```

## Initialize EyeJS

At the beginning of your `test.js` or `eye.js` file, write this:
```javascript
const eye = require("eye.js");
```
